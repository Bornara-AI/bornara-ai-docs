## Overview

This is the master expense tracking log for Bornara AI's 2026 fiscal year. It replaces the
external Excel spreadsheet with an in-repo markdown version for version control and
auditability. Use this alongside the
[Expense Tracking Guide](expense-tracking-guide.md) for detailed recording rules and the
[CRA Compliance Guide](cra-compliance-guide.md) for documentation requirements.

## Expense Log — 2026

### Column Reference

| Column          | Description                                               |
|-----------------|-----------------------------------------------------------|
| Date            | Date of expense (YYYY-MM-DD)                              |
| Category        | Expense category (from T2125 mapping)                     |
| Description     | What was purchased or paid                                |
| Amount          | Total amount paid                                         |
| Deductible Amt  | Amount deductible (some are partial, e.g., meals at 50%)  |
| T2125 Line      | Which T2125 line it maps to                               |
| Receipt?        | Yes / No — is the receipt on file?                        |
| Payment Method  | Cash / E-transfer / Credit card / PayPal                  |
| Notes           | Additional context                                        |
